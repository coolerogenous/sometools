# 第三章：核心答疑与避坑指南（新人最关心的Q&A）

## 关于试用期

### 公司说试用期不交社保，转正后补交，合法吗？

**明确答案：不合法**

根据《中华人民共和国社会保险法》规定：
- 用人单位应当自用工之日起30日内为其职工向社会保险经办机构申请办理社会保险登记
- 试用期包含在劳动合同期限内，因此试用期必须缴纳社保
- 公司以试用期为由拒绝缴纳社保属于违法行为

**维权建议：**
- 与公司HR沟通，要求正常缴纳社保
- 如沟通无效，可向当地社保监察部门投诉（电话：12333）
- 也可向劳动争议仲裁委员会申请仲裁

## 关于跳槽与断缴

### 换工作时，社保断缴一个月有什么影响？

**主要影响：**

1. **买房资格**
   - 大部分城市要求连续缴纳社保一定年限（如北京5年、上海5年）
   - 断缴会导致资格清零，需重新计算年限

2. **买车资格**
   - 部分城市（如北京、上海）要求连续缴纳社保一定年限才能摇号
   - 断缴会影响摇号资格

3. **医保报销**
   - 断缴期间无法享受医保报销待遇
   - 重新缴纳后，通常需要等待1-3个月才能恢复报销功能

4. **养老保险**
   - 养老保险累计计算，断缴一个月影响较小
   - 但长期断缴会影响退休后的养老金水平

5. **生育保险**
   - 部分城市要求连续缴纳一定时间才能享受生育津贴
   - 断缴可能影响生育待遇

### 如何无缝衔接社保？

**离职和入职时间的卡点技巧：**

1. **离职时间选择**
   - 尽量在当月15号之后离职（部分公司15号为社保缴纳截止日）
   - 这样原公司会为你缴纳当月社保

2. **入职时间选择**
   - 尽量在当月15号之前入职新公司
   - 这样新公司可以为你缴纳当月社保

3. **空档期处理**
   - 如果有较长的空档期，可考虑以灵活就业人员身份自行缴纳社保
   - 或找社保代缴机构暂时挂靠

4. **社保转移**
   - 新公司入职后，及时办理社保转移手续
   - 确保社保缴费记录连续

## 关于看病与报销

### 感冒发烧去医院，医保怎么报销？

**报销流程：**

1. **就诊前**
   - 携带社保卡到定点医院就诊
   - 挂号时出示社保卡

2. **就诊时**
   - 医生开具处方和检查单
   - 缴费时出示社保卡，直接结算

3. **报销比例**
   - 一级医院：报销比例约90%
   - 二级医院：报销比例约80%
   - 三级医院：报销比例约70%
   - 具体比例以当地政策为准

4. **注意事项**
   - 必须在医保定点医院就诊
   - 必须使用医保目录内的药品和检查项目
   - 超过起付线（通常为100-300元）后才能报销

### 看牙齿、配眼镜能刷医保吗？

**看牙齿：**
- **可以报销的项目**：
  - 补牙（包括基本材料、治疗费）
  - 拔牙
  - 治疗牙周病、牙龈炎等
- **不能报销的项目**：
  - 洗牙
  - 镶牙
  - 牙齿矫正
  - 烤瓷牙
  - 种植牙

**配眼镜：**
- **不能直接刷医保**：配眼镜属于美容类消费，不在医保报销范围内
- **特殊情况**：如果是因工伤或职业病导致的视力问题，可能可以通过工伤保险报销

## 关于公积金

### 不买房，公积金里面的钱能取出来交房租吗？怎么操作？

**可以提取**，具体操作步骤：

1. **提取条件**
   - 在缴存城市无自有住房
   - 正在租房居住
   - 连续缴存公积金满3个月

2. **提取金额**
   - 每月提取金额不超过实际房租支出
   - 不同城市有具体限额（如北京每月不超过1500元）

3. **提取材料**
   - 身份证原件及复印件
   - 住房公积金联名卡
   - 租房合同原件及复印件
   - 房租发票（部分城市需要）

4. **提取方式**
   - **线上提取**：通过公积金APP、网上办事大厅申请
   - **线下提取**：携带材料到公积金管理中心办理

5. **提取频率**
   - 通常每季度可提取一次
   - 部分城市支持按月提取

## 关于年度退税

### 每年3-6月的"个税年度汇算清缴"是什么？

**个税年度汇算清缴**是指：
- 居民个人将一个纳税年度内取得的工资薪金、劳务报酬、稿酬、特许权使用费等四项所得合并后按年计算全年最终应纳的个人所得税
- 再减除纳税年度已预缴的税款后，计算应退或应补税额
- 向税务机关办理申报并进行税款结算的行为

**为什么要进行汇算清缴？**
- 多退少补：预缴税款与实际应纳税额可能存在差异
- 确保税负公平：综合考虑各项收入和扣除

### 怎么操作才能拿回退税？

**操作步骤：**

1. **准备工作**
   - 下载并登录"个人所得税"APP
   - 检查个人信息是否正确
   - 核对专项附加扣除是否已填报

2. **开始申报**
   - 在APP首页点击"综合所得年度汇算"
   - 选择申报年度（如2025年度）
   - 选择"申报表预填服务"

3. **核对信息**
   - 核对收入和扣除信息
   - 确认专项附加扣除是否完整
   - 如有遗漏，可补充填报

4. **计算退税**
   - APP会自动计算应退或应补税额
   - 如显示"应退税额"，则可以申请退税

5. **提交申报**
   - 确认无误后，点击"提交申报"
   - 选择退税银行卡（需为本人银行卡）

6. **等待退税**
   - 税务机关审核通过后，退税会打入绑定的银行卡
   - 一般需要1-3个工作日到账

**常见退税情形：**
- 年度综合所得收入不足6万元但已预缴税款
- 有符合条件的专项附加扣除但未及时填报
- 因年中就业、退职等原因导致前后税收优惠政策不一致
- 没有任职受雇单位，仅取得劳务报酬、稿酬等所得

**注意事项：**
- 汇算清缴时间为每年3月1日至6月30日
- 如需要补税，应在6月30日前补缴，否则会产生滞纳金
- 如实申报，避免虚假申报导致的法律责任
