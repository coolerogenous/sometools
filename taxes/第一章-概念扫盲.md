# 第一章：概念扫盲（先把钱算明白）

## 什么是"五险一金"？

五险一金是指中国职工在就业时需要缴纳的社会保险和住房公积金，包括：

### 五险

1. **养老保险**
   - 为退休后提供基本生活保障
   - 个人缴纳8%，单位缴纳16-20%
   - 必须缴纳，属于强制性保险

2. **医疗保险**
   - 用于看病就医的费用报销
   - 个人缴纳2%，单位缴纳6-10%
   - 必须缴纳，属于强制性保险

3. **失业保险**
   - 失业时可领取失业金
   - 个人缴纳0.5%，单位缴纳0.5-1%
   - 必须缴纳，属于强制性保险

4. **工伤保险**
   - 工作期间发生意外伤害的保障
   - 个人不缴纳，单位全额缴纳
   - 必须缴纳，属于强制性保险

5. **生育保险**
   - 女职工生育期间的医疗费用和生育津贴
   - 个人不缴纳，单位全额缴纳
   - 必须缴纳，属于强制性保险

### 一金

6. **住房公积金**
   - 用于购房、租房、装修等住房相关支出
   - 个人和单位各缴纳5-12%（具体比例由公司确定）
   - 属于福利性质，但大部分公司都会缴纳

## 哪些是必须交的？哪些是福利性质的？

**必须交的（强制性的）：**
- 养老保险
- 医疗保险
- 失业保险
- 工伤保险
- 生育保险

**福利性质的：**
- 住房公积金（虽然不是强制，但绝大多数正规公司都会缴纳）

## 个人所得税（个税）是什么？

### 起征点（5000元）的概念

个人所得税的起征点是每月5000元，也就是说：
- 月收入不超过5000元：不需要缴纳个税
- 月收入超过5000元：超过部分需要按照税率表缴纳个税

### 为什么到手工资（税后）比合同工资（税前）少？

因为从税前工资到到手工资，需要扣除以下项目：

```
税前工资（合同工资）
  ↓
- 个人社保缴纳部分（养老8% + 医疗2% + 失业0.5% = 10.5%）
  ↓
- 个人公积金缴纳部分（5-12%）
  ↓
- 个人所得税
  ↓
= 到手工资（实发工资）
```

举个例子：
- 合同工资：10000元
- 社保扣除：10000 × 10.5% = 1050元
- 公积金扣除：10000 × 8% = 800元
- 应纳税所得额：10000 - 5000 - 1050 - 800 = 3150元
- 个税：3150 × 3% = 94.5元
- 到手工资：10000 - 1050 - 800 - 94.5 = 8055.5元

所以，10000元的税前工资，到手只有约8055.5元。

## 一分钟看懂工资条

标准工资条的结构：

```
┌─────────────────────────────────────┐
│          工资条示例                 │
├─────────────────────────────────────┤
│ 基本工资：      10000.00 元         │
│                                      │
│ 扣除项目：                          │
│ - 养老保险：    -800.00 元  (8%)    │
│ - 医疗保险：    -200.00 元  (2%)    │
│ - 失业保险：    -50.00 元   (0.5%)  │
│ - 公积金：      -800.00 元  (8%)    │
│ - 个人所得税：  -94.50 元          │
│                                      │
│ 实发工资：      8055.50 元          │
└─────────────────────────────────────┘
```

**计算公式：**
```
实发工资 = 基本工资 - 个人社保扣除 - 个人公积金扣除 - 个人所得税
```

**注意事项：**
- 工资条上的"基本工资"可能包含多个项目（基本工资、绩效、奖金等）
- 不同公司的工资条格式可能略有差异，但核心项目基本一致
- 建议每月仔细核对工资条，确保扣款准确无误
