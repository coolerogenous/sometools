# 第二章：新人必做（实操流程）

## 入职第一周：你需要配合HR做什么？

### 提供什么资料

入职新公司时，HR通常会要求提供以下资料：

1. **身份证明**
   - 身份证原件及复印件（通常需要2-3份）
   - 户口本复印件（部分公司需要）

2. **学历证明**
   - 最高学历毕业证书原件及复印件
   - 学位证书原件及复印件（如有）
   - 学信网学历验证报告（部分公司需要）

3. **银行卡**
   - 用于发放工资的银行卡（通常需要提供银行卡正反面复印件）
   - 注意：需确认公司要求的银行类型（如工商银行、建设银行等）

4. **照片**
   - 一寸免冠照片（蓝底或白底，通常需要2-4张）
   - 电子版照片（部分公司需要用于制作工作证）

5. **其他资料**
   - 离职证明（从上一家公司开具）
   - 体检报告（部分公司需要）
   - 无犯罪记录证明（部分公司需要）
   - 职业资格证书（如有，相关岗位需要）

### 入职手续办理流程

1. **提交资料**：将准备好的资料交给HR
2. **填写表格**：填写入职登记表、劳动合同等
3. **签署文件**：签署劳动合同、保密协议等
4. **领取物品**：领取工作证、办公用品等
5. **参加培训**：参加公司入职培训

## 个税APP下载与填报（重点！）

### 下载个税APP

1. **下载方式**
   - 应用商店搜索："个人所得税"
   - 扫描二维码：可通过税务局官网或公司HR提供的二维码下载

2. **注册登录**
   - 注册：使用手机号注册，设置密码
   - 登录：使用注册的手机号和密码登录
   - 实名认证：需要进行人脸识别或银行卡认证

### 填报专项附加扣除

**专项附加扣除是合法省税的利器**，包括以下六项：

1. **子女教育**
   - 扣除标准：每个子女每月1000元
   - 适用条件：子女年满3岁至博士研究生教育阶段
   - 填报材料：子女身份信息

2. **继续教育**
   - 学历（学位）继续教育：每月400元，最长48个月
   - 职业资格继续教育：取得证书当年一次性扣除3600元
   - 填报材料：教育信息或证书信息

3. **大病医疗**
   - 扣除标准：超过15000元的部分，在80000元限额内据实扣除
   - 适用条件：医保目录范围内的医药费用支出
   - 填报材料：医疗服务收费票据原件或复印件

4. **住房贷款利息**
   - 扣除标准：每月1000元，最长240个月
   - 适用条件：首套住房贷款利息
   - 填报材料：贷款合同、还款凭证

5. **住房租金**
   - 扣除标准：
     - 一线城市：每月1500元
     - 二线城市：每月1100元
     - 其他城市：每月800元
   - 适用条件：在主要工作城市无自有住房
   - 填报材料：租赁合同

6. **赡养老人**
   - 扣除标准：
     - 独生子女：每月2000元
     - 非独生子女：每月分摊2000元（每人不超过1000元）
   - 适用条件：被赡养人年满60周岁
   - 填报材料：被赡养人身份信息

### 填报步骤

1. **打开个税APP**，进入"专项附加扣除填报"
2. **选择扣除年度**（通常为当前年度）
3. **选择扣除项目**（如子女教育、住房租金等）
4. **填写相关信息**（按照APP提示填写）
5. **提交申报**：选择"通过扣缴义务人申报"（由公司在发放工资时扣除）或"综合所得年度自行申报"（次年汇算清缴时扣除）

## 社保卡的申领与激活

### 去哪里领卡？

1. **首次申领**
   - 单位统一办理：由公司HR统一收集资料办理
   - 个人办理：携带身份证到当地社保服务中心办理

2. **领取地点**
   - 公司领取：HR会通知领取时间
   - 社保服务中心：自行办理的需到指定窗口领取

### 如何激活？

#### 1. 激活医保账户

- **方法一**：到定点医院或药店使用社保卡就医或买药
- **方法二**：到社保服务中心窗口激活

#### 2. 激活金融账户

- **操作步骤**：携带社保卡和身份证到银行柜台激活
- **功能**：金融账户可用于领取养老金、失业金等社保待遇，也可作为普通银行卡使用

### 电子社保卡怎么绑定

#### 1. 微信绑定

1. 打开微信，进入"我" → "服务" → "城市服务"
2. 选择"社保" → "电子社保卡"
3. 按照提示进行人脸识别认证
4. 绑定成功后，可在微信卡包中查看

#### 2. 支付宝绑定

1. 打开支付宝，搜索"电子社保卡"
2. 进入电子社保卡小程序
3. 按照提示进行人脸识别认证
4. 绑定成功后，可在支付宝首页查看

### 社保卡使用注意事项

1. **妥善保管**：社保卡是重要的身份证明和就医凭证
2. **及时挂失**：如丢失，应立即挂失并补办
3. **定期查询**：可通过社保APP或官网查询社保缴费记录
4. **注意有效期**：社保卡有有效期，到期前需更换
