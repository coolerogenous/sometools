# 网页分析器（Web Analyzer）

## 技能概述

本技能使用 Chrome DevTools MCP 工具来访问、分析网页，并生成结构化的 Markdown 文档。适用于网站分析、项目评估、内容提取、竞品分析等场景。

## 核心能力

### 1. 网页访问与数据获取
- 使用 Chrome DevTools 打开指定网页
- 获取页面快照（基于可访问性树）
- 截取页面截图
- 执行 JavaScript 获取额外信息

### 2. 内容分析
- 提取页面标题、描述、关键词
- 识别主要内容和结构
- 提取链接、图片、表格等元素
- 分析页面布局和导航结构

### 3. 深度分析
- 多页面浏览（跟随链接）
- 技术栈识别（框架、库、工具）
- 性能指标分析（使用 Lighthouse）
- SEO 和可访问性评估

### 4. 文档生成
- 生成结构化的 Markdown 文档
- 包含目录、摘要、详细分析
- 支持自定义输出格式和内容
- 可添加截图和图表

## 工作流程

### 标准分析流程

```
1. 接收用户需求
   ↓
2. 打开目标网页
   ↓
3. 获取页面快照和截图
   ↓
4. 分析页面内容
   ↓
5. 深度探索（如需要）
   ↓
6. 生成 Markdown 文档
   ↓
7. 保存或展示文档
```

### 详细步骤

#### 步骤 1：需求确认
- 确认目标 URL
- 了解分析目的（项目评估、内容提取、竞品分析等）
- 确定输出要求（详细程度、关注重点）

#### 步骤 2：网页访问
```javascript
// 使用 Chrome DevTools MCP
mcp_chrome-devtools_new_page(url)
```

#### 步骤 3：数据收集
```javascript
// 获取页面快照
mcp_chrome-devtools_take_snapshot()

// 截取页面截图
mcp_chrome-devtools_take_screenshot()

// 执行 JavaScript 获取额外信息
mcp_chrome-devtools_evaluate_script({
  function: "() => { return document.title; }"
})
```

#### 步骤 4：内容分析
- 分析页面结构（标题、段落、列表）
- 提取关键信息（联系方式、价格、特性等）
- 识别技术栈（通过 HTML、脚本、样式）
- 评估用户体验和设计

#### 步骤 5：深度探索（可选）
- 点击导航链接探索更多页面
- 使用搜索功能查找特定内容
- 分析网络请求和性能
- 运行 Lighthouse 审计

#### 步骤 6：文档生成
生成包含以下部分的 Markdown 文档：
- 标题和元数据
- 执行摘要
- 详细分析
- 结论和建议
- 附录（截图、代码片段等）

## 命令和参数

### 主要命令

| 命令 | 描述 | 参数 |
|------|------|------|
| `analyze` | 分析单个网页 | `url`, `depth`, `focus` |
| `compare` | 对比多个网页 | `urls[]`, `criteria` |
| `extract` | 提取特定信息 | `url`, `selectors` |
| `audit` | 网站审计 | `url`, `checks[]` |

### 参数说明

#### `analyze` 命令参数
- `url`（必需）：目标网页 URL
- `depth`（可选）：分析深度（1-5，默认 1）
- `focus`（可选）：关注重点（overview, tech, content, performance, all）

#### `compare` 命令参数
- `urls[]`（必需）：要对比的 URL 列表
- `criteria`（可选）：对比标准（features, pricing, design, tech, all）

#### `extract` 命令参数
- `url`（必需）：目标网页 URL
- `selectors`（可选）：CSS 选择器或提取规则

#### `audit` 命令参数
- `url`（必需）：目标网页 URL
- `checks[]`（可选）：检查项目列表（seo, performance, accessibility, security, all）

## 限制和注意事项

### 技术限制
- 需要安装 Chrome Beta 浏览器
- 某些网站可能有反爬虫机制
- JavaScript 渲染的内容需要额外处理
- 动态加载内容可能需要等待

### 内容限制
- 无法访问需要登录的内容
- 无法绕过付费墙
- 某些地区可能有访问限制
- 实时数据可能不准确

### 性能限制
- 大型网站分析可能需要较长时间
- 深度分析会消耗更多资源
- 多页面分析需要合理规划

## 故障排除

### 常见问题

**问题 1：无法打开网页**
- 检查 URL 是否正确
- 确认网络连接
- 检查网站是否可访问

**问题 2：页面内容不完整**
- 等待页面完全加载
- 检查是否有 JavaScript 错误
- 尝试刷新页面

**问题 3：截图失败**
- 确认 Chrome Beta 已安装
- 检查页面是否已完全加载
- 尝试调整窗口大小

**问题 4：提取信息不准确**
- 检查 CSS 选择器是否正确
- 验证页面结构是否变化
- 使用多个方法交叉验证
